# WriterExtnd — домашнее задание по стилистической классификации

Этот репозиторий подготовлен как аккуратная сдача домашнего задания: в нем оставлены **ноутбук с кодом** и **документ с отчетом**, а также все ключевые артефакты экспериментов.

## Что проверять в первую очередь

- `WriterExtnd_p75.ipynb` — основной Jupyter-ноутбук с кодом, экспериментами и результатами.
- `WriterExtnd_Documentation (1).docx` — отчет/документация по проделанной работе.

## Структура репозитория

### Основные артефакты

- `WriterExtnd_p75.ipynb` — ноутбук с пайплайном и анализом.
- `WriterExtnd_Documentation (1).docx` — текстовый отчет.
- `submission.csv` — файл с предсказаниями.
- `model_params.json` — параметры модели.
- `tokenizer.pkl` — сериализованный токенизатор.

### Данные

- `dataset.zip` — исходный/подготовленный датасет.
- `authors_data/authors_data.zip` — дополнительные данные по авторам.

### Визуализации и диагностические графики

Репозиторий содержит графики для анализа данных, обучения и качества модели, включая:

- распределения классов и объема корпуса;
- кривые обучения;
- матрицу ошибок и метрики по классам;
- анализ длины фрагментов и богатства словаря;
- визуализации интерпретируемости (LIME).

## Быстрый просмотр

1. Откройте `WriterExtnd_p75.ipynb` в Jupyter/Colab для просмотра кода и результатов.
2. Откройте `WriterExtnd_Documentation (1).docx` для отчета.
3. При необходимости скачайте архивы данных (`dataset.zip`, `authors_data/authors_data.zip`) и повторите запуск.

## Примечание

Репозиторий оформлен для проверки преподавателем по ссылке и содержит все обязательные материалы: ноутбук с кодом и документ с отчетом.
